색감 - innocent사이트를 보면 red만을 써서 눈이 아프고 너무 강렬하다 -> 한국인들이 innocent에 관심을
가지는 대부분은 가족의 건강이나 아이를 생각하는 부모(여자)일 것 -> 야채 과일 채소의 색감을 다방면으로
사용하고 포인트 색상으로 innocent red를 사용

폰트 - 어느정도 그냥봐도 알수있는 부분은 놔두고 긴 문장을 번역 -> 기존 사이트에서도 장난식의 말투나
대화하는 듯한 뉘앙스 말투를 그대로 번역해서 한국어로도 대화하는 듯하게 번역 -> 폰트도 마찬가지로
아기자기한 느낌의 폰트인데 대화하는 느낌을 살리기 위해 대부분 사이트에서 사용하지 않는 손글씨체를
사용하되 최대한 지저분한 느낌은 주지않도록 노력
